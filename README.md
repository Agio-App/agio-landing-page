### Local development

1. Install dependencies:
   - `npm install`
2. Start the dev server:
   - `npm run dev`

### Build & preview

- **Build**: `npm run build`
- **Preview build locally**: `npm run preview`

### Deploy notes

- `public/_redirects` is included for static hosting providers (e.g. Netlify).
